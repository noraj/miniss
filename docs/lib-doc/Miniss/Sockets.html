    class Miniss::Sockets

    • Miniss::Sockets
    • Reference
    • Object

    Overview

    Sockets object has properties containing arrays of Sockets from the parsed /proc/net/xxx files.

    Example:

    sockets = Miniss::Sockets.new
sockets.tcpv4.each do |so|
  puts "#{so.laddr} <-> #{so.raddr}"
end
