    class Miniss::Socket

    • Miniss::Socket
    • Reference
    • Object

    Overview

    Socket object, having several properties.

    Example:

    so = Miniss::Socket.new(:tcp, 4_u8)
line = File.readlines("/proc/net/tcp")[1]
so.parse_line(line)
so.laddr # => "127.0.0.53:53"
so.state # => "LISTEN"
so.uname # => "systemd-resolve"
so.uid # => 980

    TODO IPv6 and UDP support.
